📝 Todo List App
A modern, feature-rich Todo List application built with HTML, CSS, and JavaScript.
Created and maintained by Satyam, aspiring Full Stack Developer.

🚀 Features
- Add, edit, and delete tasks
- Mark tasks as completed with checkboxes
- Persistent storage using localStorage
- Due dates with color-coded urgency:
- 🔴 Overdue tasks
- 🟠 Due today
- 🟢 Upcoming tasks
- Filter tasks by All / Active / Completed
- Sort tasks by due date
- Clear all completed tasks with one click
- Progress bar showing completion percentage
- Fully mobile responsive design
📂 Project Structure
├── index.html   # Main HTML file
├── style.css    # Styling (responsive + deadline colors)
├── index.js     # Core JavaScript logic
└── README.md    # Project documentation
🛠️ Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Storage: Browser localStorage
- Design: Responsive layout with Flexbox and media queries
📱 Mobile Responsiveness
- Inputs and buttons scale down on smaller screens
- Tasks stack vertically for readability
- Progress bar adapts to screen width

🎯 Future Enhancements
- User authentication & cloud sync
- Export tasks as PDF/CSV
- Notifications for upcoming deadlines
- Backend integration (Node.js + Express + MongoDB)
